About MyTweets

MyTweets is a mini social media project built using the Django framework. It allows users to create, view, edit, and delete tweets.

Features:

  • User registration and authentication
  • Create, edit, and delete tweets
  • Upload images with tweets
  • Sort tweets by time or title

Technologies Used:

  • Python & Django
  • HTML, CSS, Bootstrap
  • SQLite (default DB)
